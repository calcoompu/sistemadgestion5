🧪 Prueba de Sistema Anti-Duplicados

✅ VALIDACIONES IMPLEMENTADAS:

  • 📞 Código telefónico duplicado: Detecta si +XX ya existe
  • 🏳️ Nombre de país duplicado: Detecta nombres similares (case-insensitive)
  • 🔢 Formato de código: Valida formato +XXX (solo números)
  • 🔄 Sincronización: Agrega automáticamente en ambos selectores
  • 🛡️ Prevención errores: Resetea selector si hay error

⚠️ CASOS DE PRUEBA PARA VALIDAR:

  1. Duplicado por código: Intenta agregar "+54" (Argentina ya existe)
  2. Duplicado por nombre: Intenta agregar "uruguay" (si Uruguay ya existe)
  3. Código inválido: Intenta usar "54" sin el "+"
  4. Código con letras: Intenta usar "+5AB"
  5. País válido nuevo: Agrega "+352" (Luxemburgo)
